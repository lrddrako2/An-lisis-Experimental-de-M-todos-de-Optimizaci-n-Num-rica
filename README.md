# An-lisis-Experimental-de-M-todos-de-Optimizaci-n-Num-rica
Estudio comparativo de métodos de optimización de primer orden (GD, SGD, Nesterov, Adam) aplicados a funciones cuadráticas. Análisis del impacto de la convexidad, fuerte convexidad y condicionamiento numérico en la convergencia. Evaluación experimental bajo distintas tasas de aprendizaje y configuraciones matriciales.
